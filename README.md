# JavaScript DOM Practice Projects

A collection of small JavaScript projects I built while learning DOM manipulation, APIs, and UI logic. Each task lives in its own folder under **Pages/** and focuses on a specific concept.
These are learning exercises, not production apps. The goal was to get comfortable with core JavaScript patterns before jumping into frameworks.

[Live](https://santhoshv-js.vercel.app/)
